프로젝트 전체 참여자 전체 세션 파일 관리 설정
섹션 1

개요

TSL 연구 앱에서 세션 데이터를 서버로 전송하는 REST API입니다. 측정된 변수 데이터를 JSON 형태로 전송하면 프로젝트의 참여자 측정 기록에 자동으로 저장됩니다.

Base URL

https://[your-domain]/api/sessions
섹션 2

인증

모든 요청에 X-API-Key 헤더를 포함해야 합니다. 아래 API 키를 사용하세요.

{{ api_key }}
API 키를 앱 코드에 하드코딩하지 마세요. 환경 변수나 보안 저장소를 사용하세요.
섹션 3

엔드포인트

POST /api/sessions

요청 헤더

헤더값설명
Content-Type application/json 필수
X-API-Key {{ api_key }} 필수 — 위 API 키 사용

요청 본문 (JSON)

{
  "project_id":     "uuid-string",          // 필수 — 프로젝트 UUID
  "participant_id": "uuid-string",          // 필수 — 참여자 UUID
  "phase":          "사전",                 // 선택 — 예: '사전', '사후', '추적'
  "notes":          "메모 내용",            // 선택 — 자유 텍스트
  "data": {
    "variable_name":     42.5,              // 변수명: 숫자값
    "another_variable":  31.0
  }
}

응답

성공 시 200 OK

{"ok": true, "id": "새로 생성된 세션 ID"}

오류 시

{"error": "오류 메시지"}
섹션 4

코드 예시

import requests

API_KEY    = "{{ api_key }}"
BASE_URL   = "https://[your-domain]/api/sessions"

payload = {
    "project_id":     "your-project-uuid",
    "participant_id": "your-participant-uuid",
    "phase":          "사전",
    "notes":          "첫 번째 측정",
    "data": {
        "hrv": 42.5,
        "stress_score": 31.0,
    }
}

response = requests.post(
    BASE_URL,
    json=payload,
    headers={
        "X-API-Key": API_KEY,
        "Content-Type": "application/json",
    }
)

if response.status_code == 200:
    data = response.json()
    print(f"저장 완료 — 세션 ID: {data['id']}")
else:
    print(f"오류: {response.json().get('error')}")
import Foundation

let apiKey  = "{{ api_key }}"
let baseURL = URL(string: "https://[your-domain]/api/sessions")!

var request = URLRequest(url: baseURL)
request.httpMethod = "POST"
request.setValue("application/json", forHTTPHeaderField: "Content-Type")
request.setValue(apiKey,            forHTTPHeaderField: "X-API-Key")

let body: [String: Any] = [
    "project_id":     "your-project-uuid",
    "participant_id": "your-participant-uuid",
    "phase":          "사전",
    "data": ["hrv": 42.5, "stress_score": 31.0]
]
request.httpBody = try? JSONSerialization.data(withJSONObject: body)

URLSession.shared.dataTask(with: request) { data, response, error in
    guard let data = data,
          let json = try? JSONSerialization.jsonObject(with: data) as? [String: Any]
    else { return }
    if let id = json["id"] as? String {
        print("저장 완료 — 세션 ID: \(id)")
    }
}.resume()
